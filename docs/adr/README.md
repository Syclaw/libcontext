# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the libcontext project, following the format described by [Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).

Each ADR documents a significant technical decision: the context that motivated it, the alternatives considered, the decision taken, and its consequences.

Gaps in numbering (003, 005–007) are from ADRs that were drafted then either consolidated into [ADR-002's Implementation Decisions section](002-skill-plus-cli-as-primary-integration.md#implementation-decisions) or removed as premature during early development. Numbers are never reused.
